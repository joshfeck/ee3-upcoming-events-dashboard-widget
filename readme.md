ee3-upcoming-events-dashboard-widget
================

A custom upcoming events widget for Event Espresso 3

Instructions/How to use:

1. Download zip file

2. Upload via the WordPress plugins uploaded (or unzip and FTP to wp-content/plugins/)

3. Activate

More information here:

https://github.com/joshfeck/ee3-upcoming-events-dashboard-widget

Not compatible with Event Espresso 4
